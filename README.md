## 🌟 Project Overview

This project is a **fully interactive Sales Analysis Dashboard** built using **Microsoft Excel** with:

- Power Pivot  
- Pivot Tables  
- Pivot Charts  
- Data Modeling  
- Slicers & KPIs  

It helps analyze **sales performance, customer behavior, product trends, and delivery efficiency**.

---

## 🧠 Key Business Insights

✨ Total Revenue: **$3,520,984**  
👤 Avg Customer Spending: **$3,520.98**  
🚚 Avg Delivery Time: **5.53 Days**  
📉 Correlation: **0.0035 (No strong relationship detected)**
